# Contributing

## Development flow

1. Install Rust and Node.js prerequisites.
2. Run analyzer and server tests before pushing changes.
3. Run the Web UI test suite before frontend changes land.
4. Keep deterministic analysis usable without any AI provider.

## Standards

- Prefer small, evidence-backed rule changes.
- Keep issue schemas stable across analyzer, API, and UI.
- Treat heuristic findings as heuristics in both code and copy.
